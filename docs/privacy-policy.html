Privacy Policy

Gmail Subscription Manager — Chrome Extension

Effective date: February 25, 2026

What this extension does

Gmail Subscription Manager scans your Gmail inbox to identify subscription and newsletter emails, then lets you unsubscribe, archive, or whitelist them directly from the extension.

Data collection and storage

This extension does not collect, transmit, or share any personal data with external servers. All processing happens entirely on your device.

  • Email metadata (sender name, sender email, subject line, date, unsubscribe headers) is read via the Gmail API and stored locally in chrome.storage.local on your device.
  • Email bodies are never read or stored.
  • OAuth tokens are cached in chrome.storage.session (memory-only, cleared when the browser closes) and are never sent to any server other than Google's API.
  • No analytics, tracking, or telemetry of any kind is included.

Gmail API usage

The extension requests the following Google OAuth scopes:

  • gmail.readonly — to scan message headers and identify subscriptions
  • gmail.modify — to archive emails (remove the INBOX label)
  • gmail.send — to send unsubscribe emails when a subscription only supports mailto-based unsubscription

These permissions are used solely for the features described above. The extension complies with the Google API Services User Data Policy, including the Limited Use requirements.

Third-party services

The extension makes requests to:

  • Google Gmail API — to read message metadata, archive emails, and send unsubscribe messages
  • Google Favicon service — to display sender icons (google.com/s2/favicons)
  • Unsubscribe endpoints — when you click "Unsubscribe", the extension sends a request to the unsubscribe URL provided in the email's List-Unsubscribe header. These URLs are controlled by the original email sender.

No data is sent to any other third party.

Data retention and deletion

All data is stored locally on your device. You can delete all extension data at any time by:

  • Revoking access via the extension's "Sign out" button
  • Removing the extension from Chrome (Settings > Extensions)
  • Revoking the app's access at myaccount.google.com/permissions
